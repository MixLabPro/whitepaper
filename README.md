Mixlab无界社区是容纳多学科人才的跨界、创新、开源、实验性、去中心化的非盈利社区。这里汇集了科技与设计多元领域的跨学科人才。以去中心化的方式，探索新知识、新领域，挖掘个人潜能，促使个体与个体之间产生更多有价值的连接，从而产生跨领域的交汇新方向。Mixlab也是人工智能x设计跨学科探索实验场，希望能让每个人都站在各自热爱的世界里闪光。
